GUI 구현 화면

본 프로젝트는 Java Swing을 활용하여 GUI 기반 시간표 관리 프로그램으로 구현하였다.

기존 콘솔 기반 프로그램을 개선하여 사용자가 버튼과 입력창을 이용해 보다 직관적으로 시간표를 관리할 수 있도록 설계하였다.

GUI 주요 구성 요소

  • 과목명 입력
  • 요일 입력
  • 시작 시간 입력
  • 종료 시간 입력
  • 강의실 입력
  • 시간표 등록 버튼
  • 전체 조회 버튼
  • 요일별 조회 버튼
  • 수정 버튼
  • 삭제 버튼
  • 파일 저장 버튼
  • 파일 불러오기 버튼

구현 결과

사용자는 GUI 화면을 통해 시간표 등록, 조회, 수정, 삭제 기능을 수행할 수 있다.

또한 파일 저장 및 불러오기 기능을 통해 프로그램 종료 후에도 데이터를 유지할 수 있도록 구현하였다.

기대 효과

GUI 환경을 적용함으로써 사용 편의성을 향상시켰으며, Java Swing 프로그래밍에 대한 이해도를 높일 수 있었다.
