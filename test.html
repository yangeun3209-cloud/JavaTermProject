테스트 결과

프로젝트 구현 후 시간표 등록, 조회, 수정, 삭제, 정렬, 알림, 파일 저장 및 불러오기 기능이 정상적으로 동작하는지 통합 테스트를 진행하였다.

테스트 항목

기능 테스트 내용 결과
시간표 등록 과목명, 요일, 시간, 강의실 입력 후 등록 확인 정상
전체 조회 등록된 모든 시간표 출력 확인 정상
요일별 조회 입력한 요일에 해당하는 시간표만 출력 확인 정상
시간표 수정 선택한 시간표의 정보 변경 확인 정상
시간표 삭제 선택한 시간표 삭제 확인 정상
시간순 정렬 요일과 시작 시간 기준 정렬 확인 정상
수업 알림 현재 요일과 시간 기준 예정 수업 출력 확인 정상
파일 저장 시간표 데이터가 파일로 저장되는지 확인 정상
파일 불러오기 저장된 파일을 다시 불러오는지 확인 정상

예외 처리 테스트

  • 등록된 시간표가 없을 경우 안내 메시지 출력 확인
  • 잘못된 번호 입력 시 오류 메시지 출력 확인
  • 중복된 요일과 시간 입력 시 등록 차단 확인
  • 파일이 없을 경우 오류 메시지 출력 확인

테스트 결과

전체 기능은 정상적으로 동작하였으며, 잘못된 입력값에 대해서도 프로그램이 종료되지 않고 안내 메시지를 출력하도록 구현하였다.

이를 통해 프로그램의 안정성과 기본 기능 동작을 확인할 수 있었다.
