# 📚 시간표 알림 프로그램

학생들이 개인 시간표를 효율적으로 관리하고, 수업 시작 전에 알림을 받아 지각을 방지할 수 있도록 지원하는 Java 기반 시간표 관리 프로그램입니다.

---

# 01. Project Overview

## 프로젝트명
시간표 알림 프로그램

## 프로젝트 소개
시간표 알림 프로그램은 사용자가 직접 수업 정보를 등록하고 조회, 수정, 삭제할 수 있는 Java 콘솔 기반 애플리케이션입니다.

요일별 시간표 조회, 시간순 정렬, 파일 저장 및 불러오기 기능을 제공하며, 수업 시작 전에 알림을 제공하여 효율적인 일정 관리를 지원합니다.

## 프로젝트 목적
- 학생들의 시간표 관리 편의성 향상
- 수업 시작 전 알림 기능 제공
- Java 객체지향 프로그래밍 학습
- 파일 입출력 활용 경험
- GitHub 협업 경험 습득

## 기대 효과
- 일정 관리 효율 향상
- 수업 지각 방지
- 객체지향 설계 경험 확보
- 협업 프로젝트 경험 향상

# 02. Requirements Analysis

## 기능적 요구사항
- 시간표 등록
- 시간표 전체 조회
- 요일별 시간표 조회
- 시간표 수정
- 시간표 삭제
- 시간순 정렬
- 수업 시작 전 알림 기능
- 시간표 파일 저장
- 시간표 파일 불러오기

## 비기능적 요구사항
- Java 콘솔 기반 프로그램
- 직관적인 메뉴 기반 사용자 인터페이스 제공
- 객체지향 설계를 통한 유지보수성 확보
- 파일 입출력을 통한 데이터 저장 기능 지원
- GitHub를 활용한 협업 및 버전 관리
