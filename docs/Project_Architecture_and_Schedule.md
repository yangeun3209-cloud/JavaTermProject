# Project Architecture and Schedule

## 1. 프로젝트 개요

### 프로젝트명
시간표 알림 프로그램

### 프로젝트 목적
학생들이 수업 시간표를 효율적으로 관리하고, 수업 시작 전에 알림을 받아 지각을 예방할 수 있도록 지원하는 Java 기반 시간표 관리 프로그램을 개발한다.

### 프로젝트 설명
본 프로젝트는 사용자가 직접 시간표 정보를 등록하고 조회, 수정, 삭제할 수 있는 콘솔 기반 Java 애플리케이션이다.

요일별 조회, 시간순 정렬, 파일 저장 및 불러오기 기능을 제공하여 시간표 관리의 편의성을 높이고, 수업 시작 전 알림 기능을 통해 효율적인 일정 관리를 지원한다.

---

## 2. 프로젝트 구조 설계

### 2-1. 기능 구조

본 프로젝트는 다음과 같은 기능으로 구성된다.

- 시간표 등록
- 시간표 전체 조회
- 요일별 조회
- 시간표 수정
- 시간표 삭제
- 시간순 정렬
- 수업 시작 전 알림 기능
- 시간표 파일 저장
- 시간표 파일 불러오기
- 입력 예외 처리
- 중복 시간 검사
