프로젝트 개요

시간표 알림 프로그램은 학생들이 수업 시간, 강의실, 요일 정보를 보다 효율적으로 관리할 수 있도록 개발한 Java Swing GUI 기반 프로그램이다.

기존 콘솔 기반 프로그램을 GUI 환경으로 개선하여 사용자가 보다 직관적으로 사용할 수 있도록 구현하였다. 사용자는 과목명, 요일, 시작 시간, 종료 시간, 강의실 정보를 등록할 수 있으며, 조회, 수정, 삭제 기능을 통해 시간표를 체계적으로 관리할 수 있다.

또한 파일 저장 및 불러오기 기능을 제공하여 프로그램을 종료한 후에도 기존 시간표 데이터를 유지할 수 있도록 구현하였다.

프로젝트 목적

학생들은 여러 과목의 수업 시간과 강의실 정보를 직접 기억해야 하는 경우가 많다. 본 프로그램은 시간표 정보를 체계적으로 관리하고 필요한 정보를 쉽게 확인할 수 있도록 지원하는 것을 목적으로 한다.

기대 효과

  • 개인 시간표 관리 편의성 향상
  • 수업 시간 및 강의실 확인 지원
  • 수업 일정 관리 효율성 향상
  • Java 객체지향 프로그래밍 학습
  • GUI 프로그래밍 경험 습득
