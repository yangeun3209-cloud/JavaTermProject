📚 시간표 알림 프로그램

Java Swing GUI 기반 시간표 관리 프로그램

팀장 : 양은찬 | 팀원 : 우민지

프로젝트 소개

학생들이 개인 시간표를 효율적으로 관리하고 수업 시작 전에 알림을 받을 수 있도록 제작한 Java Swing GUI 프로그램입니다.

주요 기능

  • 시간표 등록
  • 시간표 전체 조회
  • 요일별 조회
  • 시간표 수정
  • 시간표 삭제
  • 시간순 정렬
  • 수업 알림
  • 파일 저장
  • 파일 불러오기

클래스 구조

클래스 역할
Main 프로그램 시작
Subject 과목 정보 저장
ScheduleManager 시간표 관리
TimetableApp GUI 및 이벤트 처리

사용 기술

  • Java
  • Swing GUI
  • IntelliJ IDEA
  • GitHub
  • ObjectInputStream / ObjectOutputStream
