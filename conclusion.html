프로젝트 결론

시간표 알림 프로그램은 학생들이 수업 시간과 강의실 정보를 보다 효율적으로 관리할 수 있도록 개발한 Java Swing GUI 기반 프로그램이다.

시간표 등록, 조회, 수정, 삭제, 정렬, 수업 알림, 파일 저장 및 불러오기 기능을 구현하여 실제로 사용할 수 있는 형태의 프로그램을 완성하였다.

프로젝트 성과

  • Java 객체지향 프로그래밍 적용
  • Java Swing GUI 구현
  • ArrayList를 활용한 데이터 관리
  • 파일 입출력 기능 구현
  • 예외 처리 기능 적용
  • GitHub를 활용한 프로젝트 관리
  • GitHub Pages 발표 페이지 제작

배운 점

이번 프로젝트를 통해 Java 프로그램을 설계하고 구현하는 과정을 경험할 수 있었다.

특히 클래스 분리를 통한 객체지향 설계 방법과 파일 입출력, 예외 처리의 중요성을 이해할 수 있었다.

또한 GitHub를 활용하여 프로젝트를 관리하고 버전을 관리하는 방법을 학습할 수 있었다.

개선 방향

  • 데이터베이스 연동 기능 추가
  • 모바일 알림 기능 추가
  • 사용자 계정 기능 추가
  • 시간표 자동 추천 기능 추가
  • 캘린더 연동 기능 추가

최종 결론

본 프로젝트를 통해 Java GUI 프로그래밍과 객체지향 설계 능력을 향상시킬 수 있었으며, 실제 사용 가능한 시간표 관리 프로그램을 구현하는 경험을 얻을 수 있었다.

향후 다양한 기능을 추가하여 더욱 완성도 높은 시간표 관리 프로그램으로 발전시킬 수 있을 것으로 기대한다.
