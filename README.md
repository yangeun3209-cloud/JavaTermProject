# 📚 시간표 알림 프로그램

학생들이 개인 시간표를 쉽게 관리하고, 수업 시작 전에 알림을 받아 지각을 줄일 수 있도록 도와주는 Java 기반 시간표 관리 프로그램

---

## 📌 프로젝트 소개

시간표 알림 프로그램은 사용자가 수업 정보를 직접 등록하고 조회하며, 수업 시작 전에 알림을 받을 수 있도록 제작한 콘솔 기반 Java 애플리케이션입니다.

요일별 시간표 관리와 파일 저장 기능을 통해 효율적인 학습 일정 관리를 지원합니다.

---

## ✨ 주요 기능

### 📖 시간표 관리
- 수업 등록 / 조회 / 수정 / 삭제
- 요일별 시간표 조회
- 시간순 자동 정렬
- 강의실 정보 저장

### ⏰ 알림 기능
- 수업 시작 전 알림 시간 설정
- 콘솔 알림 메시지 출력

### 💾 파일 저장 기능
- 시간표 파일 저장
- 저장된 시간표 불러오기

### ⚠ 예외 처리
- 잘못된 입력 처리
- 중복 시간 검사
- 시간 형식 검증
