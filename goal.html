개발 목표

본 프로젝트의 목표는 학생들이 시간표를 보다 쉽고 효율적으로 관리할 수 있는 프로그램을 개발하는 것이다.

기존에는 수업 시간과 강의실 정보를 직접 확인해야 했지만, 본 프로그램을 통해 시간표를 등록하고 관리할 수 있도록 구현하였다.

세부 목표

  • 학생들의 시간표 관리 편의성 향상
  • 수업 일정 확인 시간 단축
  • 수업 시작 전 알림 기능 제공
  • Java 객체지향 프로그래밍 학습
  • Java Swing GUI 프로그래밍 경험 습득
  • 파일 저장 및 불러오기 기능 구현
  • 예외 처리를 통한 안정적인 프로그램 개발
  • GitHub를 활용한 프로젝트 관리 경험 습득

개발 방향

사용자가 쉽게 사용할 수 있는 GUI 환경을 제공하는 것을 우선 목표로 설정하였다. 또한 유지보수와 기능 확장을 고려하여 클래스별 역할을 분리하는 객체지향 설계를 적용하였다.

기대 효과

시간표 정보를 체계적으로 관리할 수 있으며, Java 프로그래밍 기술과 GitHub 협업 경험을 향상시킬 수 있다.
