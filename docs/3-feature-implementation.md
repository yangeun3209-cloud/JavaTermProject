# Feature Implementation

## 1. 기능 구현 개요

시간표 알림 프로그램은 Java 콘솔 환경에서 동작하며, 사용자가 메뉴를 선택하여 시간표를 관리할 수 있도록 구현하였다.

---

## 2. 시간표 등록

사용자로부터 과목명, 요일, 시작 시간, 종료 시간, 강의실 정보를 입력받아 시간표 목록에 추가한다.

구현 내용

- Subject 객체 생성
- ArrayList에 시간표 저장
- 중복 시간 검사 적용

---

## 3. 시간표 전체 조회

등록된 모든 시간표를 출력한다.

구현 내용

- 저장된 시간표 목록 출력
- 등록된 시간표가 없을 경우 안내 메시지 출력

---

## 4. 요일별 조회

사용자가 입력한 요일에 해당하는 수업만 출력한다.

구현 내용

- 입력된 요일과 저장된 요일 비교
- 해당 요일 수업만 출력
- 결과가 없을 경우 안내 메시지 출력

---

## 5. 시간표 수정

등록된 시간표 번호를 선택하여 새로운 정보로 수정한다.

구현 내용

- 수정할 번호 입력
- 새로운 과목 정보 입력
- 기존 데이터 교체
- 잘못된 번호 입력 시 오류 안내

---

## 6. 시간표 삭제

등록된 시간표 번호를 선택하여 삭제한다.

구현 내용

- 삭제할 번호 입력
- ArrayList에서 해당 데이터 삭제
- 잘못된 번호 입력 시 오류 안내

---

## 7. 시간순 정렬

요일과 시작 시간을 기준으로 시간표를 정렬한다.

구현 내용

- Comparator 사용
- 요일 기준 정렬
- 시작 시간 기준 정렬

---

## 8. 수업 알림

현재 요일과 시간을 입력받아 이후 예정된 수업을 알려준다.

구현 내용

- 현재 요일 입력
- 현재 시간 입력
- 이후 수업 정보 출력

---

## 9. 파일 저장 및 불러오기

시간표 데이터를 파일로 저장하고 다시 불러올 수 있도록 구현하였다.

구현 내용

- ObjectOutputStream을 활용한 파일 저장
- ObjectInputStream을 활용한 파일 불러오기
- 파일 오류 예외 처리

---

## 10. 구현 결과

시간표 등록, 조회, 수정, 삭제, 정렬, 알림, 파일 저장 및 불러오기 기능을 구현하였다.

이를 통해 사용자가 콘솔 환경에서 시간표를 효율적으로 관리할 수 있도록 하였다.
