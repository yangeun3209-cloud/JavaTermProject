# Project Architecture and Schedule

# 1. 프로젝트 개요

## 프로젝트명
시간표 알림 프로그램

## 프로젝트 소개
시간표 알림 프로그램은 학생들이 개인 시간표를 효율적으로 관리하고, 수업 시작 전에 알림을 받아 지각을 방지할 수 있도록 지원하는 Java 기반 콘솔 애플리케이션입니다.

사용자가 직접 수업 정보를 등록, 조회, 수정, 삭제할 수 있으며 파일 저장 및 불러오기 기능을 통해 시간표 데이터를 관리할 수 있습니다.

## 프로젝트 목적
- 학생들의 시간표 관리 편의성 향상
- 수업 시작 전 알림 기능 제공
- Java 객체지향 프로그래밍 학습
- 파일 입출력 기능 활용 경험
- GitHub 협업 프로젝트 경험 습득

# 4. GitHub 협업 관리

## 협업 방식
- GitHub Repository를 활용한 버전 관리
- 기능별 역할 분담 후 개별 구현
- 구현 완료 후 Pull Request 또는 병합 진행
- 통합 테스트를 통해 최종 기능 검증

## GitHub 관리 방식
- GitHub Issues를 활용한 기능별 작업 등록
- 담당자(Assignee) 지정으로 역할 분담
- Milestone을 활용한 일정 관리
- Pull Request를 통한 코드 병합 및 검토

## 의사소통 방식
- 카카오톡을 통한 실시간 소통
- 대면 회의를 통한 진행 상황 공유
- 문제 발생 시 즉시 협의 후 해결
