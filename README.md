# 시간표 알림 프로그램

## 프로젝트 소개

학생들이 개인 시간표를 효율적으로 관리하고 수업 시작 전에 알림을 받을 수 있도록 지원하는 Java 콘솔 기반 시간표 관리 프로그램이다.

시간표 등록, 조회, 수정, 삭제 기능뿐만 아니라 시간순 정렬, 수업 알림, 파일 저장 및 불러오기 기능을 제공한다.

## 개발 환경

* Java

## 주요 기능

* 시간표 등록
* 시간표 전체 조회
* 요일별 조회
* 시간표 수정
* 시간표 삭제
* 시간순 정렬
* 수업 알림
* 파일 저장
* 파일 불러오기
* 입력 예외 처리
* 중복 시간 검사

---

## 실행 방법

1. 프로젝트 다운로드
2. IntelliJ IDEA에서 프로젝트 열기
3. Main.java 실행
4. 메뉴를 선택하여 기능 사용
