# Summary

## 1. 프로젝트 요약

시간표 알림 프로그램은 학생들이 개인 시간표를 효율적으로 관리하고 수업 시작 전에 알림을 받을 수 있도록 지원하는 Java 콘솔 기반 프로그램이다.

---

## 2. 프로젝트 목적

수업 요일, 시간, 강의실 정보를 체계적으로 관리하여 학생들이 수업 일정을 쉽게 확인할 수 있도록 하는 것을 목표로 하였다.

---

## 3. 주요 기능

- 시간표 등록
- 시간표 전체 조회
- 요일별 조회
- 시간표 수정
- 시간표 삭제
- 시간순 정렬
- 수업 알림
- 파일 저장
- 파일 불러오기

---

## 4. 구현 결과

Java의 클래스, 객체, ArrayList, 파일 입출력, 예외 처리 기능을 활용하여 시간표 관리 프로그램을 구현하였다.

프로그램은 메뉴 기반 콘솔 UI로 구성되어 있으며, 사용자가 원하는 기능을 선택하여 시간표를 관리할 수 있다.

---

## 5. 느낀 점

이번 프로젝트를 통해 Java 객체지향 프로그래밍의 기본 구조를 이해할 수 있었다.

또한 GitHub를 활용하여 프로젝트 문서와 소스 코드를 관리하는 방법을 학습하였다.

기능별로 클래스를 분리하면서 코드의 가독성과 유지보수성이 중요하다는 것을 알게 되었다.
