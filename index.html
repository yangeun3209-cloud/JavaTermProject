📚 시간표 알림 프로그램

1. 프로젝트 개요

시간표 알림 프로그램은 학생들이 수업 시간, 요일, 강의실 정보를 효율적으로 관리할 수 있도록 제작한 Java Swing GUI 기반 프로그램이다.

기존 콘솔 프로그램을 GUI 형태로 개선하여 사용자가 보다 직관적으로 시간표를 관리할 수 있도록 구현하였다.

2. 개발 목표

  • 시간표 관리 편의성 향상
  • 수업 일정 확인 지원
  • 수업 시작 전 알림 기능 제공
  • 파일 저장 및 불러오기 기능 제공
  • Java 객체지향 프로그래밍 학습

3. 주요 기능

  • 시간표 등록
  • 시간표 전체 조회
  • 요일별 조회
  • 시간표 수정
  • 시간표 삭제
  • 시간순 정렬
  • 수업 알림
  • 파일 저장
  • 파일 불러오기
  • 입력 예외 처리
  • 중복 시간 검사

4. 클래스 구조

클래스 역할
Main 프로그램 실행 시작점
TimetableApp GUI 화면 및 이벤트 처리
ScheduleManager 시간표 데이터 관리
Subject 시간표 정보 저장

5. GUI 구현 화면

Java Swing을 활용하여 GUI 기반 시간표 관리 프로그램을 구현하였다.

사용자는 입력창과 버튼을 통해 시간표를 등록, 조회, 수정, 삭제할 수 있다.

6. 핵심 구현 내용

시간표 등록

과목명, 요일, 시작 시간, 종료 시간, 강의실 정보를 입력받아 저장한다.

조회 기능

전체 조회 및 요일별 조회 기능을 제공한다.

수정 및 삭제

등록된 시간표를 선택하여 수정 및 삭제할 수 있다.

시간순 정렬

요일과 시작 시간을 기준으로 정렬한다.

수업 알림

현재 요일과 시간을 기준으로 예정된 수업을 안내한다.

파일 입출력

ObjectOutputStream과 ObjectInputStream을 활용하여 시간표 저장 및 불러오기 기능을 구현하였다.
