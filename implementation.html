핵심 구현 내용

시간표 알림 프로그램은 Java 객체지향 프로그래밍 개념을 활용하여 구현하였다. 데이터 저장, 관리, 사용자 인터페이스 기능을 각각 분리하여 유지보수성과 확장성을 높였다.

1. 시간표 등록 기능

사용자가 입력한 과목명, 요일, 시작 시간, 종료 시간, 강의실 정보를 Subject 객체로 생성하여 저장한다.

등록된 데이터는 ArrayList를 이용하여 관리하며, 여러 개의 시간표를 저장할 수 있도록 구현하였다.

2. 중복 시간 검사

동일한 요일과 시간에 이미 등록된 수업이 존재하는 경우 중복 등록을 방지하도록 구현하였다.

이를 통해 사용자가 실수로 같은 시간의 수업을 여러 번 등록하는 문제를 해결하였다.

3. 조회 기능

전체 조회 기능을 통해 등록된 모든 시간표를 확인할 수 있다.

요일별 조회 기능을 이용하면 특정 요일의 수업만 출력할 수 있도록 구현하였다.

4. 시간표 수정 및 삭제

사용자는 등록된 시간표를 선택하여 수정하거나 삭제할 수 있다.

잘못된 번호를 입력한 경우 오류 메시지를 출력하여 프로그램이 종료되지 않도록 처리하였다.

5. 시간순 정렬

Comparator를 활용하여 시간표를 요일과 시작 시간 기준으로 정렬하였다.

정렬 기능을 통해 사용자가 시간표를 보다 쉽게 확인할 수 있도록 구현하였다.

6. 파일 저장 및 불러오기

ObjectOutputStream을 사용하여 시간표 데이터를 파일에 저장하였다.

ObjectInputStream을 사용하여 저장된 데이터를 다시 불러올 수 있도록 구현하였다.

이를 통해 프로그램을 종료한 후에도 시간표 정보를 유지할 수 있다.

7. 예외 처리

숫자가 아닌 값을 입력하거나 존재하지 않는 번호를 입력한 경우 예외 처리를 적용하였다.

예외 발생 시 프로그램이 종료되지 않고 안내 메시지를 출력하도록 구현하였다.

구현 결과

시간표 등록, 조회, 수정, 삭제, 정렬, 파일 저장 및 불러오기 기능을 모두 구현하였으며, 사용자가 GUI 환경에서 시간표를 효율적으로 관리할 수 있도록 개발하였다.
