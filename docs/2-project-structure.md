# Project Architecture and Schedule

## 1. 프로젝트 개요

### 프로젝트명

시간표 알림 프로그램

### 프로젝트 목적

학생들이 개인 시간표를 효율적으로 관리하고 수업 시작 전에 알림을 받아 지각을 방지할 수 있도록 지원하는 Java 기반 시간표 관리 프로그램 개발

---

## 2. 프로젝트 구조 설계

### 2-1. 기능 구조

* 시간표 등록
* 시간표 전체 조회
* 요일별 조회
* 시간표 수정
* 시간표 삭제
* 시간순 정렬
* 수업 시작 전 알림
* 시간표 파일 저장
* 시간표 파일 불러오기
* 입력 예외 처리
* 중복 시간 검사

### 2-2. 화면 구조

```text
===== 시간표 알림 프로그램 =====

1. 시간표 등록
2. 시간표 전체 조회
3. 요일별 조회
4. 시간표 수정
5. 시간표 삭제
6. 시간순 정렬
7. 수업 알림
8. 파일 저장
9. 파일 불러오기
0. 종료
```

### 2-3. 시스템 구조

```text
사용자
   ↓
TimetableApp
   ├── 시간표 등록
   ├── 시간표 조회
   ├── 수정
   ├── 삭제
   ├── 정렬
   ├── 알림
   ├── 저장
   └── 불러오기
            ↓
     ScheduleManager
            ↓
         Subject
```

### 2-4. 프로젝트 구조도

```text
src/
├── Main.java
├── Subject.java
├── ScheduleManager.java
└── TimetableApp.java
```

## 4. 협업 및 의사소통 방안

### 협업 방식

* GitHub Repository를 이용한 버전 관리
* 기능별 역할 분담 후 개별 구현
* 구현 완료 후 코드 병합
* 통합 테스트 진행

### 의사소통 방식

* 카카오톡을 통한 실시간 소통
* 대면 회의를 통한 진행 상황 공유
* 문제 발생 시 즉시 협의 및 해결
