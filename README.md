# 시간표 알림 프로그램

## 1. 프로젝트 소개

### 프로젝트명
시간표 알림 프로그램

### 프로젝트 목적
학생들이 수업 시간표를 쉽게 확인하고, 수업 시작 전 알림을 받을 수 있도록 도와주는 자바 기반 프로그램입니다.

### 기대 효과
- 수업 시간을 놓치는 상황 감소
- 개인 시간표 관리 편의성 향상
- Java 객체지향 프로그래밍 및 파일 입출력 학습
- 팀 프로젝트 협업 능력 향상

---

## 2. 프로젝트 배경 및 범위

### 배경
대학생들은 요일마다 수업 시간이 다르고 강의실도 달라 혼동하는 경우가 많습니다.  
또한 바쁜 일상 속에서 수업 시간을 놓치는 경우도 발생합니다.  
이 프로젝트는 시간표를 등록하고 수업 시작 전에 알림을 제공하여 효율적인 학사 일정 관리를 돕기 위해 기획되었습니다.

### 포함 기능
- 수업 정보 등록
- 요일별 시간표 조회
- 수업 시간 수정 및 삭제
- 수업 시작 전 알림 기능
- 시간표 파일 저장 및 불러오기

### 제외 기능
- 모바일 앱 푸시 알림
- 학교 포털과 자동 연동
- 외부 데이터베이스 연동
- 위치 기반 알림 서비스

### 사용 기술
- Java
- IntelliJ IDEA
- GitHub
