# Project Structure

## 1. 프로젝트 구조 설계

시간표 알림 프로그램은 Java 콘솔 기반 프로그램으로, 기능별 역할을 나누어 클래스를 구성하였다.

---

## 2. 클래스 구조

| 클래스명 | 역할 |
|---|---|
| Main | 프로그램 실행 시작점 |
| Subject | 시간표 데이터 저장 |
| ScheduleManager | 시간표 관리 기능 처리 |
| TimetableApp | 사용자 메뉴 및 입력 처리 |

---

## 3. 기능 구조

- 시간표 등록
- 시간표 전체 조회
- 요일별 조회
- 시간표 수정
- 시간표 삭제
- 시간순 정렬
- 수업 알림
- 파일 저장
- 파일 불러오기
- 예외 처리
- 중복 시간 검사

## 7. 구조 설계 결과

프로젝트를 기능별 클래스로 분리하여 코드의 가독성과 유지보수성을 높였다.

또한 GitHub 문서와 소스 코드를 함께 관리하여 프로젝트 진행 과정을 명확하게 정리하였다.
