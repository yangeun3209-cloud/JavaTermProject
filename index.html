📚 시간표 알림 프로그램

1. 프로젝트 개요

학생들이 개인 시간표를 효율적으로 관리하고 수업 시작 전에 알림을 받을 수 있도록 제작한 Java Swing GUI 프로그램입니다.

2. 개발 목표

  • 시간표 관리 편의성 향상
  • 수업 일정 확인 지원
  • 수업 시작 전 알림 기능 제공

3. 주요 기능

  • 시간표 등록
  • 전체 조회
  • 요일별 조회
  • 수정 및 삭제
  • 시간순 정렬
  • 수업 알림
  • 파일 저장 및 불러오기

4. 클래스 구조

  • Main
  • TimetableApp
  • ScheduleManager
  • Subject

5. GUI 구현 화면

Java Swing을 이용하여 GUI 환경으로 구현하였습니다.

6. 핵심 구현 내용

  • ArrayList를 이용한 시간표 관리
  • 중복 시간 검사
  • 예외 처리
  • 파일 입출력 기능 구현

7. 테스트 및 결과

등록, 조회, 수정, 삭제, 정렬, 저장 및 불러오기 기능을 모두 테스트하였으며 정상 동작을 확인하였습니다.

8. GitHub 관리

README 작성, docs 문서 작성, Issues 관리 및 GitHub Pages를 활용하여 프로젝트를 관리하였습니다.

9. Release

최종 버전 v1.0.0 Release 생성 완료

10. 마무리

Java 객체지향 프로그래밍과 GitHub 협업 및 프로젝트 관리 방법을 학습할 수 있었습니다.
