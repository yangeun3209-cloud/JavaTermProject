# 프로젝트 구조 및 역할·일정 관리 계획

## 프로젝트명
📚 시간표 알림 프로그램

---

## 1. 프로젝트 개요

시간표 알림 프로그램은 학생들이 개인 시간표를 효율적으로 관리하고, 수업 시작 전에 알림을 받아 지각을 방지할 수 있도록 돕는 Java 기반 콘솔 애플리케이션입니다.

사용자는 수업 정보를 등록, 조회, 수정, 삭제할 수 있으며, 알림 기능과 파일 저장/불러오기 기능을 통해 시간표를 체계적으로 관리할 수 있습니다.

## 2. 프로젝트 구조 설계

---

### 2-2. 프로젝트 구조 설명

본 프로젝트는 객체지향 구조를 기반으로 기능별 클래스를 분리하여 설계하였다.

- **Main**
    - 프로그램 실행 및 메인 메뉴 출력
    - 사용자 입력 처리

- **ScheduleManager**
    - 시간표 등록, 조회, 수정, 삭제
    - 요일별 조회
    - 시간표 정렬
    - 중복 시간 검사
    - 입력 예외 처리

- **AlarmManager**
    - 수업 시작 전 알림 기능 처리

- **FileManager**
    - 시간표 저장 및 불러오기 기능 처리

- **Subject**
    - 시간표 데이터 저장 클래스
