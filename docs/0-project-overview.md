# Project Overview

## 1. 프로젝트 개요

프로젝트명: 시간표 알림 프로그램

본 프로젝트는 학생들이 개인 시간표를 효율적으로 관리하고 수업 시작 전에 알림을 받을 수 있도록 지원하는 Java 콘솔 기반 프로그램이다.

---

## 2. 프로젝트 목적

학생들은 여러 과목의 수업 시간, 요일, 강의실 정보를 직접 기억해야 하는 경우가 많다.

본 프로그램은 시간표 정보를 등록하고 조회할 수 있도록 하여 수업 일정 관리의 편의성을 높이는 것을 목적으로 한다.

---

## 3. 주요 기능

- 시간표 등록
- 시간표 전체 조회
- 요일별 조회
- 시간표 수정
- 시간표 삭제
- 시간순 정렬
- 수업 알림
- 파일 저장
- 파일 불러오기
- 입력 예외 처리
- 중복 시간 검사

---

## 4. 기대 효과

- 개인 시간표 관리 편의성 향상
- 수업 시간 확인 지원
- 수업 시작 전 알림 기능 제공
- Java 객체지향 프로그래밍 학습
- 파일 입출력 기능 학습
